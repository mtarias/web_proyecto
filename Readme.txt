Aqui le subimos un readme explicando las cosas importantes de la pagina web.

- Para ver el contenido el usuario debe loguearse
	- Solo los eventos publicos pueden verse en el main
- Aparecen los eventos que el usuario crea, ademas de poder crear eventos
	- Solo se puede crear eventos con el usuario logueado
- Los eventos aparecen ordenados ya sea por fecha (Eventos anteriores, proximos eventos) o si son publicos (Eventos publicos)
- El evento aparece con el estilo que pusimos en el mock-up
- Se pueden invitar a usuarios logueados a traves de un buscador
	- El usuario aparece automaticamente en la lista de invitados
- Se pueden subir fotos (pero aun no implementamos que se muestren en la pagina)
- Al crear un nuevo evento, guarda al usuario como administrador
- Se puede cerrar sesion
- Se muestra la lista de invitados (ya sea si es admin o es un invitado)
- Arregladas las ventanas con css basico
- Se pueden agregar comentarios al evento
	- Aparece con el nombre de quien lo creo
